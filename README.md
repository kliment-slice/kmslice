# kmSLICE

## RFTs: requests for tracks 🎵

Want me to play something? Open a PR.

## How to request a track

1. **Fork** and **Star** this repo
2. **Edit** [`REQUESTS.md`](REQUESTS.md) — add a row to the table:
   ```
   | Requestor | Artist  | Track | Comment |
   | --------- | ------- | ----- | ------- |
   ```
3. **Open a PR**

That's it. I'll play it.
